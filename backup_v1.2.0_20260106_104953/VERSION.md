# 快人快语 (FastVoice) v1.2.0

## 版本信息
- **版本号**: v1.2.0
- **发布日期**: 2026-01-06
- **版本类型**: 功能更新版本

## 🎯 核心更新

### 1. 双模式快捷键系统
实现了灵活的快捷键触发模式：

#### 一次按键模式（语音输入）
- **快捷键**: 左 Option (⌥)
- **交互**: 按下开始录音，松开停止
- **尾音收集**: 200ms

#### 双击+长按模式（翻译）
- **快捷键**: 右 Option (⌥)
- **交互**: 轻按→长按确认→松开停止
- **尾音收集**: 200ms

### 2. 左右修饰键识别
完整支持左右修饰键独立识别

### 3. 配置驱动设计
完全根据配置文件动态监听

### 4. 按键监听测试工具
新增 test_key_listener.py
