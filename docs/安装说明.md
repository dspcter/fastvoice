# FastVoice 安装指南

## 安装步骤

### 1. 安装应用
```bash
# 将应用复制到应用程序文件夹
cp -R FastVoice.app /Applications/

# 或者直接在 Finder 中拖拽
```

### 2. 首次打开
由于应用未签名，首次打开需要：
1. 在 Finder 中找到 `/Applications/FastVoice.app`
2. **右键点击** → 选择「打开」
3. 点击「打开」确认
4. 之后就可以正常双击打开了

### 3. 授权权限
**麦克风权限**：
- 首次使用语音功能时，系统会提示授权麦克风
- 点击「好」允许

**辅助功能权限**：
- 快捷键需要辅助功能权限
- 打开：「系统设置」→「隐私与安全性」→「辅助功能」
- 点击「+」号，添加 `FastVoice`
- 确保已勾选

## 模型文件

### 模型位置
模型文件存储在工作目录中：
```
~/Documents/claude/快人快语/models/models/
├── asr/
│   └── sense-voice/          # ASR 模型 (~2GB)
└── translation/             # 翻译模型（可选）
    ├── marianmt-zh-en/
    └── marianmt-en-zh/
```

### 首次运行
- 应用会自动检测模型是否存在
- 如果模型不存在，会显示下载向导
- ASR 模型必需，翻译模型可选

## 卸载

### 方法 1: 使用卸载脚本
```bash
cd ~/Documents/claude/快人快语
./uninstall.sh
```

### 方法 2: 手动卸载
1. 删除应用：`/Applications/FastVoice.app`
2. 删除工作目录：`~/Documents/claude/快人快语/`

## 使用提示

1. **托盘图标**：应用运行时在右上角菜单栏显示图标
   - 双击图标：打开设置
   - 右键图标：退出应用

2. **模型下载**：首次使用需下载 ASR 模型（约 700MB）
   - 下载完成后可离线使用
   - 翻译模型可选，按需下载

3. **权限问题**：如果快捷键无反应
   - 检查辅助功能权限是否已授权
   - 重新启动应用

## 故障排除

### 应用无法打开
1. 右键点击 → 选择「打开」
2. 如果还无法打开，检查系统日志

### 快捷键无反应
1. 检查「系统设置」→「隐私与安全性」→「辅助功能」
2. 确保 FastVoice 已添加并勾选

### 模型下载失败
1. 检查网络连接
2. 重新打开应用，会自动重试
3. 查看日志文件：`~/Documents/claude/快人快语/logs/FastVoice.log`
